# **TSTP Permission Changer**

Effortlessly manage file and folder ownership and permissions with the **TSTP Permission Changer**. This robust, lightweight, multi-threaded tool is designed to save time, ensure accuracy, and provide comprehensive reporting for all operations. Whether you're a system administrator, IT professional, or power user, TSTP Permission Changer offers an intuitive interface and powerful features to streamline your file management tasks.

## **Features**

- **Comprehensive Ownership Management**  
  - Easily assign yourself or other users as the owner of files and folders.  
  - Perform recursive ownership changes, applying updates to all child directories and files seamlessly.

- **Detailed Reporting and Analytics**  
  - Generate comprehensive reports that detail changed, unchanged, and error entries for your operations.  
  - Save reports in `.txt` format or other supported file types to your preferred directory.  
  - Export reports for further analysis or record-keeping.

- **Change Reversion Capabilities**  
  - Utilize the built-in database to track original ownership and permissions, enabling easy reversion of changes when necessary.  
  - Maintain version history of permissions and ownership for auditing and compliance purposes.

- **Real-Time Progress Monitoring**  
  - View live updates of changed, unchanged, and error counts as operations progress.  
  - Enjoy a unified progress bar that enhances the user experience by providing clear status indicators.

- **Advanced Multi-Threaded Processing**  
  - Accelerate file operations with efficient multi-threaded processing, reducing operation times significantly, especially on large datasets.

- **Customizable User Interface with Dark Mode**  
  - Switch effortlessly between light and dark themes to reduce eye strain and tailor the interface to your preferences.  
  - Intuitive and user-friendly design ensures ease of navigation and operation.

- **Automated Scheduling & Task Management**  
  - Schedule permission changes and ownership updates to occur during specified times, allowing for unattended operations.  
  - Manage multiple tasks simultaneously with prioritized processing queues.

- **Security and Compliance**  
  - Ensure that all operations adhere to your organizational security policies and compliance requirements.  
  - Log all changes meticulously for audit trails and accountability.

## **How to Use**

1. **Download and Launch**:  
   Download the latest version from [https://tstp.xyz/software/permission-changer](https://tstp.xyz/software/permission-changer). Ensure you have the necessary administrator rights to execute ownership and permission changes.

2. **Install Dependencies**:  
   If required, install any necessary dependencies as outlined in the [Installation Guide](https://tstp.xyz/software/permission-changer/installation).

3. **Select Files/Folders**:  
   Launch the application and use the intuitive user interface to select individual files or folders for ownership updates or permission checks. You can also drag and drop items into the application window for quick selection.

4. **Choose Actions**:  
   - **Check Ownership Info**: View detailed information about the current owner, creation date, and modification date of selected files and folders.  
   - **Change Permissions**: Update ownership recursively across selected files and folders, ensuring consistency and compliance with your organization's policies.

5. **Configure Settings**:  
   Customize your operation preferences, such as selecting specific user accounts for ownership changes, setting recursive depth, and choosing report formats and destinations.

6. **Execute Operations**:  
   Initiate the ownership or permission changes and monitor the real-time progress through the unified progress bar and status indicators.

7. **View and Save Reports**:  
   Upon completion, review the detailed reports of operations, highlighting changed, unchanged, and error entries. Save these reports in your chosen directory for record-keeping or further analysis.

8. **Revert Changes if Necessary**:  
   Utilize the reversion feature by accessing the built-in database to restore original ownership and permissions, ensuring that any unintended changes can be easily undone.
